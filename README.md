# Descripción

**Barbery** es una aplicación web diseñada para facilitar la administración de barberías mediante la gestión de citas, clientes, barberos y servicios. La plataforma permite organizar las reservaciones, controlar los horarios disponibles y mejorar la experiencia tanto de los clientes como del personal.

---

# Problemática

Muchas barberías aún administran las citas mediante llamadas telefónicas, mensajes o agendas físicas, lo que ocasiona pérdida de información, duplicidad de reservaciones y dificultades para controlar la disponibilidad de los barberos.

Además, los clientes no cuentan con una forma rápida y sencilla de consultar los horarios disponibles o reservar una cita sin acudir al establecimiento.

---

# Solución Propuesta

Se desarrolló una plataforma web que permite:

- Registro de clientes.
- Inicio de sesión.
- Gestión de barberos.
- Administración de servicios.
- Agenda de citas.
- Control de disponibilidad de horarios.
- Administración de usuarios.
- Consulta del historial de citas.
- Gestión de reportes administrativos.

---

# Arquitectura

El sistema fue desarrollado bajo una arquitectura **MVC (Modelo-Vista-Controlador)**, permitiendo separar la lógica de negocio, la interfaz de usuario y el acceso a los datos.

La arquitectura está conformada por:

- Frontend desarrollado con HTML, CSS, JavaScript y Bootstrap.
- Backend desarrollado en ASP.NET Core con C#.
- API REST para la lógica del sistema.
- Base de datos PostgreSQL.

---

# Desarrollo del Proyecto

El proyecto inició con el análisis del proceso de reservación de citas en barberías y la identificación de las principales necesidades administrativas.

Posteriormente se diseñó la base de datos considerando entidades como clientes, barberos, servicios, citas, horarios y usuarios.

Después se implementó la lógica de negocio utilizando ASP.NET Core y se desarrollaron los diferentes módulos del sistema para administrar la información de manera eficiente.

Finalmente se realizaron pruebas funcionales para verificar el correcto registro de citas, disponibilidad de horarios y administración de usuarios.

# Resultados Obtenidos

Se desarrolló una plataforma funcional que optimiza la administración de una barbería, permitiendo gestionar citas de manera eficiente, reducir errores en la programación de horarios y mejorar la organización del negocio.

# Impacto

Barbery contribuye a la digitalización de las barberías, ofreciendo una solución moderna para la administración de citas y clientes.

El sistema mejora la organización del establecimiento, optimiza el tiempo de atención y proporciona una mejor experiencia tanto para los clientes como para los barberos.

Durante el desarrollo del proyecto se fortalecieron conocimientos en desarrollo web, arquitectura MVC, bases de datos relacionales, APIs REST y control de versiones con Git.
